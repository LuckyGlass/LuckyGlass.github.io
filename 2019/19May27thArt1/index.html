杂记 - 2019级编程社的重组

发表于 2019-05-27 更新于 2019-11-14 分类于 杂记 Valine：

暑假将至，按常理编程社将会进行一次重组
旧成员有的或许忙于中考放弃竞赛（再见），或仍坚持留在社内；又会有新成员进入社团
至于我为什么要把这篇文章公开发布，或许是希望能够真正引起大家注意吧……

· 延续

自从中考前一个月（2019.4），竞赛全部停止，同学们全身心地投入中考复习、似乎忘记了竞赛；也有些同学在此期间学习受阻，认为信息技术并非自己的出路，便默然离开了编程社（虽然我们尊重他们的选择，但我仍为他们感到遗憾）。

当然这也成了编程社历年的“洗劫”，当老友们重聚在机房，就必然会发现一些同学消失了（当然大部分仍坚持着）。

在高一入学时，照例还会招一批新成员。因为今年开始信息学竞赛的降分名额缩水，不知道招收成员会不会受影响。

· 迎新

首先需要认识到的是高一开始学习竞赛并不算晚，也有许多成功的例子。学习竞赛（特指信息学）最初必定会发现自己跟不上老学员的进度，这时候重点是习惯机房里的生活（这里将会成为接下来的两年中我们待的时间最久的地方）——重中之重，从老学员那里学到经验（不是学“如何‘颓废’而不被老师发现”），尤其是以前的一些课件，可以要来看一看。

以及一些传递信息方式，QQ群之类的一定是要加的，各种OJ有机会都注册一个账号。特别注意在QQ群里面不要乱刷跟信息学无关的玩意儿。

Welcome!

· 老学员们

众所周知，对于大多数学员来说，高二就将是竞赛的终点。我们已经真正是一个“老”学员了，留给我们发挥实力的机会不多——比起初中的同学们，我们更应该重视。

过去我们发现自己往往了解的知识甚至不如初二的大神们广泛。

因此我主张自学新知，由于一些同学们可能不知道自己学习的方向，我为2019年剩下的几个月设立了一些计划。下面是$《\mathcal June Program》$ 的主要内容：

  1. 莫队（始终念念不忘）

  2. 概率论：依照我购买的《概率论与数理统计》一书系统学习

    • 古典概型
    • 期望（包括连续期望）：需要学习微积分

  3. 线性代数：$TLY$ 推荐

  4. 复习内容：

    • Splay
    • DP
    • 网络流

希望通过此次完整的复习可以弥补之前的一些“缺失疏漏”，请各成员密切注意Vjudge上的活动（新学员也可以的）。

The End
