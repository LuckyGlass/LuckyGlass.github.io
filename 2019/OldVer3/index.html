游记 - 2019天梯赛 重大之行

发表于 2019-03-31 更新于 2019-04-07 分类于 杂记 热度： Valine：

第一次参加天梯赛 ~
作为重庆中学生联队参加；

Tab. 准备省选去了……这篇游记搁置了好久，也只能从印象中找到比赛的过程了

一、重大之行

重大的大学城校区还是非常漂亮的（词穷了），到达的时间还比较早，本可以在校园里四处逛一逛，但是天不作美，淅淅沥沥地下起了小雨。

莫名把伞落在车里，只能在科技楼（考试的场地）楼下等待。（同学到的有点晚，不知道是不是雨下大了的原因）

人到齐后互相寒暄了几句，膜了一膜高一的神犇。

开幕式先介绍了重庆赛区参赛的各大高校，最后还特别提及了一下中学生联队 QwQ 。然后主要讲了比赛的注意事项，之前在 QQ 群里也已经公布了 —— 这次会采用一种监考软件，听起来查得特别严格，而且如果被判为作弊，全校参赛队员就会当场驱逐，的确有一点丢不起这个脸。想必这一届没有什么人作弊了吧 ~

开幕式结束，午餐（大学里从科技楼走到食堂都要接近十分钟……）

二、入场

唯一的印象就是一间机房里的电脑数量大概是我所在学校的一间机房的电脑数量的两三倍……

然后选手挨着坐，两台电脑之间连隔板都没有（惊了，这么随便？）

三、开局

忍不住吐槽

现场的指导员在选手入场后就立了一个flag —— “评测网站可能会出一些问题，在最后 $15min$ 崩溃属于正常现象，但是前几届都还没有出现这种现象”。

然后一开始评测网站就出问题了……

到了比赛开始时间，比赛显示“未开始”。一些选手点入比赛看题，然后发现加载不出来……表示什么都不敢动（万一被踢出去了就不好了）

大概比赛正式开始十多分钟后才恢复正常。

四、比赛过程

题目分成了三个难度：“基础题”，“进阶题”，“登顶题”。

当然从前往后挨个刷~ “基础题”非常愉快，大概看懂题就出题解了。就不多说，完成时间是比赛开始后 $1h$ 。

然后“进阶题”自闭了一会儿……看不懂题……直接跳过了两道（而且听其他同学说还比较简单）。看到后面的进阶题还是思路比较清晰，手写了一个栈（莫名其妙STL的 stack<> 调试起来有问题）和一个动态规划。接下来就是整场比赛最揪心的一道模拟题……

我以为后面的“登顶题”很难，然后就没去看。于是死磕模拟题。大概就是给你一个由若干单词和符号组成的句子，要你把它的一些字符处理或者把一些单词处理一下输出出来。调试不出来是最心塞的……大概磨蹭了一个多小时（再也不死磕模拟题了 QwQ）

五、结束

其实我觉得我不怎么会写游记，草草收个尾吧

结果在中学生联队的我所在队伍的排名是第三……（竟然被一个NOIP提高组二等奖的超了）

在重大逛了一会再离开……

Worth visiting!

The End
