OI - CSP2019模拟赛 Day2 B

发表于 2019-10-25 分类于 OI题解 Valine：

怎么又是一道容斥……

Part1. 题面

有一个正 n 边形（n 是奇数），你要在上面选择 m 个不同顶点，使得这 m 个点围成的凸多边形 恰好 包含 k 个锐角。

给出 n,m,k；问有多少种不同的选择方案，两个方案被认为不同当且仅当两个方案所选取的顶点集合不同。

请注意，你需要回答 q 组询问（每组询问给出 n,m,k）。

数据规模：1≤q≤105,3≤m≤n≤106,0≤k≤m1\le q\le10^5,3\le m\le n\le10^6,0\le k\le m1≤q≤10​5​​,3≤m≤n≤10​6​​,0≤k≤m。

Part2. 解析

众所周知，正多边形的所有顶点共圆，而且“等弦对等角”。

Part2/1. 形成锐角的充要条件

那么可以直观地看出：在圆上依次选取 3 个点 A,B,CA,B,CA,B,C，要使 ∠ABC<90∘\angle ABC<90^\circ∠ABC<90​∘​​，充要条件 是 \frac{l}{2}< \overset{\frown}{AB}+\overset{\frown} {BC}

那么反映在正多边形上，定义 xix_ix​i​​ 表示选取的凸多边形的第 i 个顶点与第 i+1 个顶点之间隔了多少条正 n 边形的边。要使第 i 个角是锐角，则需要满足 xi−1+xi>n2x_{i-1}+x_i>\frac n2x​i−1​​+x​i​​>​2​​n​​，又因为 nnn 是奇数，可以得到 xi−1+xi≥n+12x_{i-1}+x_i\ge\frac{n+1}2x​i−1​​+x​i​​≥​2​​n+1​​ 。为了表示方便，设 p=n+12p=\frac{n+1}2p=​2​​n+1​​

比如上图（加粗的点是选取的 m 个顶点，设 1 是选取的第一个点，然后顺时针选点），则 x1=x2=x4=2,x3=1x_1=x_2=x_4=2,x_3=1x​1​​=x​2​​=x​4​​=2,x​3​​=1。那么 ∠1<90∘\angle1<90^\circ∠1<90​∘​​ 因为 x0+x1≥px_0+x_1\ge px​0​​+x​1​​≥p，\angle5\not<90^\circ 因为 x2+x3<px_2+x_3<px​2​​+x​3​​<p 。

也正因为这一点，选取的 m 边形最多只有 3 个锐角 ，且当且仅当 m=3（三角形）时，可以同时存在 3 个锐角。

Part2/2. 容斥原理

这种题目，一般来说直接计算“恰好”是非常困难的，因此要么采用“正难则反”，要么就用更加复杂的容斥原理。而显然这道题是用 容斥。

我们发现下面这些值是可以计算的：

  • ans0ans_0ans​0​​：任意选取 m 个点构成凸多边形的方案数；

  • ans1ans_1ans​1​​：对于每个角，有多少种选取 m 个点的方案中它是锐角，即
    \sum_{i=1}^n(有多少种\angle i是锐角)；

  • ans2ans_2ans​2​​：类似于 ans1ans_1ans​1​​，只是变成了两个角而已
    ans_2=\sum_{(i,j)}(有多少种方案中\angle i和\angle j都是锐角)；

  • ans3ans_3ans​3​​：仍然是这样的定义
    ans_3=\sum_{(i,j,k)}(多少种方案中\angle i,\angle j,\angle k都是锐角)

设我们要求的答案 ansi′ans_i'ans​i​′​​ 表示 恰好有 iii 个角是锐角的方案数。那么有：

  • ans0=ans0′+ans1′+ans2′+ans3′ans_0=ans_0'+ans_1'+ans_2'+ans_3'ans​0​​=ans​0​′​​+ans​1​′​​+ans​2​′​​+ans​3​′​​
    根据 ans0ans_0ans​0​​ 的定义可知。

  • ans1=ans1′+2×ans2′+3×ans3′ans_1=ans_1'+2\times ans_2'+3\times ans_3'ans​1​​=ans​1​′​​+2×ans​2​′​​+3×ans​3​′​​

    因为只有两个锐角的方案中，假设这两个锐角分别是 ∠i∠j\angle i\angle j∠i∠j ，那么计算 ∠i\angle i∠i 为锐角的方案数时，这个方案被统计了一次，计算 ∠j\angle j∠j 为锐角的方案数时，这个方案又被统计了一次。总计所有恰好有两个锐角的方案被统计了两次。

    三个锐角的方案被统计了三次，同理。

  • ans2=ans2′+3×ans3′ans_2=ans_2'+3\times ans_3'ans​2​​=ans​2​′​​+3×ans​3​′​​

    对于一个恰好有三个角的方案，假设三个锐角分别 ∠i∠j∠k\angle i\angle j\angle k∠i∠j∠k，那么计算 ∠j∠j\angle j\angle j∠j∠j 为锐角时，这个方案算了一次，∠i∠k\angle i\angle k∠i∠k 为锐角时，这个方案又算了一次，计算 ∠j∠k\angle j\angle k∠j∠k 为锐角时，这个方案还会被算一次。总计被算了 3 次。

  • ans3=ans3′ans_3=ans_3'ans​3​​=ans​3​′​​ 显然。

然后配一配容斥系数：

  • ans0′=ans0−ans1+ans2−ans3ans_0'=ans_0-ans_1+ans_2-ans_3ans​0​′​​=ans​0​​−ans​1​​+ans​2​​−ans​3​​
  • ans1′=ans1−2×ans2+3×ans3ans_1'=ans_1-2\times ans_2+3\times ans_3ans​1​′​​=ans​1​​−2×ans​2​​+3×ans​3​​
  • ans2′=ans2−3×ans3ans_2'=ans_2-3\times ans_3ans​2​′​​=ans​2​​−3×ans​3​​
  • ans3′=ans3ans_3'=ans_3ans​3​′​​=ans​3​​

Part2/3. 计算

根据 xix_ix​i​​ 的定义，可以得到下面这个方程：

{∑i=1mxi=n∀ixi≥1\begin{cases} \sum_{i=1}^mx_i=n\\ \forall _ix_i\ge 1 \end{cases} ​⎩​⎪​⎨​⎪​⎧​​​​i=1​∑​m​​x​i​​=n​∀​i​​x​i​​≥1​​

这个方程的解的个数就是 ans0ans_0ans​0​​，用隔板法即可： ans0=Cn−1m−1ans_0=C_{n-1}^{m-1}ans​0​​=C​n−1​m−1​​。

然后计算 ans1ans_1ans​1​​，不妨设 ∠t\angle t∠t 为锐角，那么 xt−1+xt≥px_{t-1}+x_t\ge px​t−1​​+x​t​​≥p。则存在两种情况：

  1. xt≥px_t\ge px​t​​≥p：那么 xt−1x_{t-1}x​t−1​​ 就没有限制，只要 xt−1≥1x_{t-1}\ge1x​t−1​​≥1 即可；接下来就是常规操作，将 xtx_tx​t​​ 的值减去 p−1p-1p−1，那么此时 xt≥1x_t\ge1x​t​​≥1，∑i=1mxi=n−p+1\sum_{i=1}^mx_i=n-p+1∑​i=1​m​​x​i​​=n−p+1；

    再用隔板法：Cn−pm−1C_{n-p}^{m-1}C​n−p​m−1​​

  2. xt<px_t<px​t​​<p：设 h=xth=x_th=x​t​​，那么 xt−1≥p−hx_{t-1}\ge p-hx​t−1​​≥p−h，再进行常规操作，将 xt−1x_{t-1}x​t−1​​ 的值减去 p−h−1p-h-1p−h−1，此时 xt−1≥1x_{t-1}\ge1x​t−1​​≥1，∑i=1mxi=n−p+h+1\sum_{i=1}^mx_i=n-p+h+1∑​i=1​m​​x​i​​=n−p+h+1；

    但是这个式子中，我们知道 xt=hx_t=hx​t​​=h，所以我们单独把 xtx_tx​t​​ 提出来，相当于：

    \begin {align} \sum_{i=1}^{m-1}x_i+x_t&=n-p+h+1\\ \sum_{i=1}^{m-1}x_i&=n-p+1 \end{align}

    发现又可以用隔板法了：Cn−pm−2C_{n-p}^{m-2}C​n−p​m−2​​。但是 hhh 本身还有 p−1p-1p−1 种取值，所以最后方案数是 (p−1)Cn−pm−2(p-1)C_{n-p}^{m-2}(p−1)C​n−p​m−2​​

对于一个 ∠t\angle t∠t 是这么多，一共有 mmm 个 ∠t\angle t∠t，综上 ans1=m×[Cn−pm−1+Cn−pm−2(p−1)]ans_1=m\times\left[C_{n-p}^{m-1}+C_{n-p}^{m-2}(p-1)\right]ans​1​​=m×[C​n−p​m−1​​+C​n−p​m−2​​(p−1)]

接下来是 ans2ans_2ans​2​​，不难发现这两个锐角一定是相邻的，不妨设是 ∠t\angle t∠t 和 ∠(t+1)\angle(t+1)∠(t+1)，则有 xt−1+xt≥p,xt+xt+1≥px_{t-1}+x_t\ge p,x_t+x_{t+1}\ge px​t−1​​+x​t​​≥p,x​t​​+x​t+1​​≥p，仍然存在两种情况：

  1. xt≥px_t\ge px​t​​≥p：和 ans1ans_1ans​1​​ 一样，先减去 p−1p-1p−1 再隔板法：Cn−pm−1C_{n-p}^{m-1}C​n−p​m−1​​；

  2. xt<px_t<px​t​​<p：令 h=xth=x_th=x​t​​，则 xt−1≥p−h,xt+1≥p−hx_{t-1}\ge p-h,x_{t+1}\ge p-hx​t−1​​≥p−h,x​t+1​​≥p−h，再给 xt−1x_{t-1}x​t−1​​ 和 xt+1x_{t+1}x​t+1​​ 都减去 p−h−1p-h-1p−h−1，又知道 xt=hx_t=hx​t​​=h，则

    \begin{align} \sum_{i=1}^{m-1}x_i+h&=n-2p+2h+2\\ \sum_{i=1}^{m-1}x_i&=n-2p+h+2 \end{align}

    惊奇地发现等式左边还有一个未知数 hhh，但是我们知道 p−h≥1p-h\ge1p−h≥1，所以可以把 (p−h)(p-h)(p−h) 整体当作一个新的未知数；

    ∑i=1m−1xi+(p−h)=n−p+2\sum_{i=1}^{m-1}x_i+(p-h)=n-p+2 ​i=1​∑​m−1​​x​i​​+(p−h)=n−p+2

    相当于 mmm 个未知数。用隔板法：Cn−p+1m−1C_{n-p+1}^{m-1}C​n−p+1​m−1​​。但是不需要乘上 (p−1)(p-1)(p−1)，因为解出来这个方程，hhh 的值也就知道了。
