Part1. From FFT to NTT

FFT虽然跑得快，但是因为是浮点数运算，终究还是有 精度、常数 等问题。 - Tiw_Air_OAO

而且处理模数还比较麻烦……

于是就引入了 NTT —— 在模的意义下（多项式的系数对某数取模）将多项式和点值表达之间等价转化。分析 FFT 必须用到 double，会卡精度的原因，发现主要是 单位复根 在大多数情况下是一个涉及浮点数的复数。

NTT 即考虑在模的意义下，用别的东西来替换掉单位复根……

Part2. 原根

对于素数 ppp，定义它的原根为 G0,G1,G2,...,Gp−2G^0,G^1,G^2,...,G^{p-2}G​0​​,G​1​​,G​2​​,...,G​p−2​​ 使得这 p−1p-1p−1 个数互不相同（其中 GGG 是整数）。

再根据原根定义出：对于一个 n−1n-1n−1 次多项式（也就是 nnn 项的多项式）， g_n^k=(G^{\frac {p-1}n})^k\bmod p（k=0 (n−1)k=0\text ~(n-1)k=0 (n−1)）。其中需要保证 p−1n\frac{p-1}n​n​​p−1​​ 是一个整数 ——
只要学过 FFT，应该会知道一定要先把多项式的项数改为一个 222 的幂才能 FFT ——
NTT 也一样，所以 nnn 很有可能是一个比较大的 222 的幂，要使得 p−1n\frac {p-1}n​n​​p−1​​ 是整数，那么 p−1p-1p−1 就需要包含很多个因子 222，但是这样的 ppp 并不多。这就造成了 NTT 的实际应用非常受限……

那么我们能否用 gnkg_n^kg​n​k​​ 来替换掉单位复根？这需要检验它在模意义下是否具有单位复根在 FFT 中的全部性质……

Part2/1. 与单位复根的相似性

在 FFT 中，我们总共用到了单位复根的这些性质：

  1. nnn 个单位复根互不相同；
  2. ωnk=ω2n2k\omega_n^k=\omega_{2n}^{2k}ω​n​k​​=ω​2n​2k​​；
  3. ωnk=−ωnk+n/2\omega_n^{k}=-\omega_n^{k+n/2}ω​n​k​​=−ω​n​k+n/2​​；
  4. ωna×ωnb=ωna+b\omega_n^a\times\omega_n^b=\omega_n^{a+b}ω​n​a​​×ω​n​b​​=ω​n​a+b​​。

于是再来检验一下 gnkg_n^kg​n​k​​ 的性质：

  1. 根据原根的定义，gnkg_n^kg​n​k​​ 也互不相同；

  2. g2n2k=(Gp−12n)2k=(Gp−1n)k=gnkg_{2n}^{2k}=(G^{\frac{p-1}{2n}})^{2k}=(G^{\frac{p-1}{n}})^k=g_n^kg​2n​2k​​=(G​​2n​​p−1​​​​)​2k​​=(G​​n​​p−1​​​​)​k​​=g​n​k​​；

  3. gnk+n/2=(Gp−1n)k+n/2=(Gp−1n)k⋅(Gp−12)=gnk⋅Gp−12g_n^{k+n/2}=(G^{\frac{p-1}n})^{k+n/2}=(G^{\frac{p-1}n})^k\cdot(G^{\frac{p-1}2})=g_n^k\cdot G^{\frac{p-1}2}g​n​k+n/2​​=(G​​n​​p−1​​​​)​k+n/2​​=(G​​n​​p−1​​​​)​k​​⋅(G​​2​​p−1​​​​)=g​n​k​​⋅G​​2​​p−1​​​​；

    因为 (Gp−12)2=Gp−1=1(G^{\frac{p-1}2})^2=G^{p-1}=1(G​​2​​p−1​​​​)​2​​=G​p−1​​=1 （费马小定理）

    然后又根据原根的定义 G^{\frac{p-1}2}\not=G^{p-1}；

    所以得到 Gp−12=−1G^{\frac{p-1}2}=-1G​​2​​p−1​​​​=−1；

    所以 gnk+n/2=−gnkg_n^{k+n/2}=-g_n^kg​n​k+n/2​​=−g​n​k​​；

  4. gna×gnb=(Gp−1n)a×(Gp−1n)b=(Gp−1n)a+b=gna+bg_n^a\times g_n^b=(G^{\frac{p-1}n})^a\times(G^{\frac{p-1}n})^b=(G^{\frac{p-1}n})^{a+b}=g_n^{a+b}g​n​a​​×g​n​b​​=(G​​n​​p−1​​​​)​a​​×(G​​n​​p−1​​​​)​b​​=(G​​n​​p−1​​​​)​a+b​​=g​n​a+b​​；

开心的发现它具有单位复根的所有性质~
