杂记 - 你将不再是我

发表于 2020-10-28 更新于 2020-10-30 分类于 杂记 热度： Valine：

我的心中那沉闷到底为谁拥有，向着天空，伸出了手——我多么希望再一次对你说……

# 小引

过去你我形影相随，共同漫过这如玻璃般透明的时光。

“然而你终究不是我啊，再见了，你将远行，去往与我不一样的方向”；

“然而你毕竟不是我啊，再见了，你将驻足，寻求与我不一样的希望”。

# “Missing_Glass”

过去我们一起支配着有限的时间，我们有着共同的目标，有着相同的信仰，甚至于相同的思想。

过去我们是如此相似……

祈求着谁 能够将我的心房轻轻叩击；今天的你 是否会留意并转身去靠近；因为或许 下个路口仍是同样的结局 不存在 刹那的 奇迹

——《世末歌者》

我们共度淋漓的世末之雨，在此般荒凉的世末之境等待着谁将自己的心房轻轻叩击。唯有温暖的人情能够抚慰世末的伤痕，但鲜有人能够为谁而驻足；他们漫无目的地奔走，在这世末的雨中被冲刷消散。你我共同欣赏着歌者嘶哑却在透凉的雨中愈发清亮的歌声，希望有谁能牵起歌者的手，打破无休止的世末循环。

那一刻，远方昏黄的落日映照着乌黑的云，世末之境铭刻在你我心中，每当瓢泼大雨，耳边都会不禁想起熟悉的旋律——欣赏世末。

最后这一杯 若是能 拂去你一生的风尘；余下的年华 停止漂泊吧 在这座空城留下

——《三杯空城》

离别时的第一杯酒，寒彻人心，一座灯火通明的城市，因你的缺失，后街已堆满残雪、巷中只剩刺骨的冰冷，如同一座死寂的空城。只能独自饮酒，耳边清风拂过，恰如离别场面，然而只剩下虚掩的木门，等待着有谁叩响。待到繁华散尽，洗去你容颜，与之一并洗去少年之事。这最后一杯，能否拂去你一生的风尘，将你留下？

为你，这城华灯初上。

所幸经年漂浮红尘中 这颗心已是千疮百孔 怎惧你以薄情为刃 添一道裂缝 又不会痛

——《我的一个道姑朋友》

我们一同经历过离合，时光倥偬，怎能不生情？

未必薄情，或许薄情的是我，每一次不道而别，第一次 2016 年，第二次 2017 年，第三次 2019 年……痛的唯有划开伤痕的人。你我一同刻下名字，祈求着不要再增添伤痛。

光与影是否不该在这一刻相逢 而你和我从来谁也不是谁的附庸

——《零和》

有光便会产生影，但毕竟光与影彼此分离，你与我的相逢不过是偶然，较为长久的偶然。

但毕竟你我谁也不是谁的附庸，也正因此，你应当离开了。

我总是以为可以如此安稳一生，顺其自然便能达成理想。然而不是的，你的观念与我完全不同；时间证明，你是对的。

我已经落后于时间，或许无法跟上你的脚步了，我们的距离渐渐拉开，如同被光撕裂为两半的影子——一半显得愈发棱角鲜明，而另一半却渐渐消逝了。

是时候收拾行装了，有些东西是我所厌恶的，但你可未必。譬如时间罢，我向来厌恶提前为时间作打算，因为计划总赶不上变化；但是你比我更会掌管时间，不愿让任何时间被临时的打算浪费，因此将这未曾用过的计划本赠予你吧。又比如绘本吧，过去绘画是我引以为豪的特长，但是现在几支铅笔都只能拿来画图像了；你比我更懂艺术的意义，那么这本余留许多页的绘本就赠予你了。

再见了，你将远行。

你是星空下的万家灯火 你也是全世界最好的过客；最怕你 某天酒意散去做离开的抉择

——《予你成歌》

尽管如此，我仍不忘当初作下陪伴的承诺，守着你的执着。

去追寻你的诗和远方吧，不必留意。我也不会停留于此，你我将前往不同的方向。或许某天你会和他人聊起这一块玻璃的故事，只不过看起来它也并不如此幸运。

# 祈明

你赐予了我太多，我是如此坚信你我将踏上同一条道路，未曾想到你也有退下舞台的一天。

万物已不屑传阅的篇章 便由我诵读至声嘶音哑；在最后一垠纯净里蜗居；待黑白分明；待黑白分明

——《象牙塔少女》

你我曾高居于象牙塔之上，与我们所憎恶的人潮势不两立。但鲜有人能够理解，象牙塔这一方渺小天地。无法改变，独善自身——独自高捧自己的信仰，待黑白分明。

终又看见某个黄昏，看见林下春风，像每个儿时眼中的梦。
而那些 至死不渝的初衷 曾牢牢握在掌中，像最灿烂星空。

终又梦见那片壮阔云海，看见万顷桃红，掀起心底的暴雨山洪。
默默 死去或默默永生 至少也曾心怀孤勇，与之温暖相拥。

——《旁观者S的独白》

那一刻，你是一个冷眼静观世间风云变幻的旁观者，因为无力改变世界而选择在角落静观其变，叹息命运万千。

大地的癔病是漆黑的癔病 我等的梦境是洁白的梦境 神明的药引是鲜红的药引；不洁由我清涤

——《灰死病少女》

但我终究不是忍耐得住的人啊，那天我背离你离开塔顶，为祈求光明。

我试图以我的思想去影响周围的人，尽管收效甚微，
