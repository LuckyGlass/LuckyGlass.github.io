# 题意

求 nnn 个节点的形态不同的（无标号）二叉树的叶子节点数量的期望。

数据规模：n≤109n\le 10^9n≤10​9​​

# 解析

看到 nnn 的规模，大概知道这是道结论题……下面来推一下结论：

从生成函数考虑，因为是无标号，应该是用普通生成函数。

设 pip_ip​i​​ 为 iii 个节点的二叉树的期望叶子数，那么就要求 pnp_np​n​​。再设 cic_ic​i​​ 表示 iii 个节点的二叉树的数量，显然有 ci=∑j=0i−1cjci−1−jc_i=\sum_{j=0}^{i-1}c_jc_{i-1-j}c​i​​=∑​j=0​i−1​​c​j​​c​i−1−j​​（枚举左右两棵子树的大小）。根据期望的线性性，可以像 DP 一样写出式子：

pi=∑j=0i−1cjci−j−1(pj+pi−j−1)cip_i=\frac{\sum_{j=0}^{i-1}c_jc_{i-j-1}(p_j+p_{i-j-1})}{c_i} p​i​​=​c​i​​​​∑​j=0​i−1​​c​j​​c​i−j−1​​(p​j​​+p​i−j−1​​)​​

这个式子的意思就是讨论根的左右子树的大小，分别为 jjj 和 (i−j−1)(i-j-1)(i−j−1)，那么概率为 cjci−j−1ci\frac{c_jc_{i-j-1}}{c_i}​c​i​​​​c​j​​c​i−j−1​​​​，对期望的贡献为 pi+pi−j−1p_i+p_{i-j-1}p​i​​+p​i−j−1​​，于是 获得该贡献的概率×贡献 就是期望了。两边同时乘上 cic_ic​i​​：

cipi=∑j=0i−1cjci−j−1(pj+pi−j−1)c_ip_i=\sum_{j=0}^{i-1}c_jc_{i-j-1}(p_j+p_{i-j-1}) c​i​​p​i​​=​j=0​∑​i−1​​c​j​​c​i−j−1​​(p​j​​+p​i−j−1​​)

观察到如果把括号展开，则有非常相似的 cxpxc_xp_xc​x​​p​x​​ 形式，于是令 tx=cxpxt_x=c_xp_xt​x​​=c​x​​p​x​​。

t_i=\sum_{j=0}^{i-1}(t_jc_{i-j-1}+t_{i-j-1}c_{j})=2\sum_{j=0}^{i-1}t_jc_{i-j-1}\\t_{i+1}=2\sum_{j=0}^it_jc_{i-j}

可以看到第二个式子是卷积的形式，也就是：

∑i=0+∞ti+1=2(∑i=0+∞ti)(∑i=0+∞ci)\sum_{i=0}^{+\infty}t_{i+1}=2\left(\sum_{i=0}^{+\infty}t_i\right)\left(\sum_{i=0}^{+\infty}c_{i}\right) ​i=0​∑​+∞​​t​i+1​​=2(​i=0​∑​+∞​​t​i​​)(​i=0​∑​+∞​​c​i​​)
