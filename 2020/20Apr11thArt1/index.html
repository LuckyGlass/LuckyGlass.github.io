杂记 - Belong to none

发表于 2020-04-11 分类于 杂记 热度： Valine：

愿能驱散所有不愿

在师长看来，学生无非两类，玩游戏的和不玩游戏的。连某些名校也以此为面试的一项指标。甚至某教练说，只要你玩游戏，那他就会把你从第一梯队踢出。

那好，再会

# GROUP Ⅰ

说实话，很羡慕也很佩服那些成天玩着也泰然自若的人，秉承着“人生得意需尽欢”的人生哲理。能够面对老师家长的责斥不屑一顾，看到不玩游戏的同学比自己学习好不心生愧疚。

虚拟世界，多好！物我无忧！

这也是我喜欢 Minecraft 的一个原因，一个新的空间，没有牛顿，不需要关心什么死生，哪怕游戏中一天几乎转瞬即逝，人物也不会衰老，哪怕死亡，最多不过重开存档再一轮回。

想要在这样的虚拟环境中探寻，却又被一些所愿以及非所愿扯回。

无法沉浸。不像能够无所顾虑地笑谈“颓废”的人。他们能全身心投入其中，虚拟中发展，虚拟中生存。

# GROUP Ⅱ

深受老师家长青睐——从来不玩游戏，要放松就看书的顶尖学霸。

praise, respect, honor, opportunity……为其所有，无可匹敌。

提到“游戏”，微微一笑，“我不反对游戏，但是我不玩游戏”，掌声随即潮涌。

“这次考试的第一，仍然是……这背后的原因是什么？这是因为别人每天都在努力。”听到赞扬，心平气和，毕竟“努力是理所应当”。

“你要是努力学习，你就像他那样；要是你还像这样，你就会像……那样。”前途无量，未到时机，其实早已确定。

# GROUP Ⅲ

好想颓废啊——不行我还有作业——好累啊，想休息——你看别人都还在学，你在干什么——自闭了，算了吧——可能多自闭一会就好了？你看……不就是这样的吗

眼观八方，能时刻注意他人举措。三人行必有我师，取其精华，去其糟粕。

去掉了别人的糟粕，自己的却始终去不掉。也不愿去掉。

自己的优点一直保持，别人的却始终学不到。也无法学到。

# Belong to NONE

时评：你要站队，你看材料怎么说的，怎么好写，就站哪个立场。

班会：同学们，看看你属于哪一个层次。

为什么不能无关——你这是骑墙。

为什么不能在中间——你得有一个准确的个人定位。

明明是一个连续区间却硬生生划分成若干个离散的点，明明无法分类，偏要草率归类。三个组不是全集，中间有不计其数的其他点。

我绝对不是不玩游戏的Ⅱ类，但是我知道，如果别人知道我玩游戏，会不由自主地划分向Ⅰ类，饱受嘲讽的一类。但是我不是Ⅰ类，我无法忍受他人的轻视，我希望得到认可，不仅仅是虚拟中一个个任务的完成，NPC的”praise”。

于是，伪装是必要的。我得尽可能在别人面前表现出“我不玩游戏”的风范。

不是Ⅲ，有时实在想玩了，能够沉浸在其中玩一下午，有时候玩久了，能够马上扔掉装成一直在学习的模样。

但是，再划出一类Ⅳ吗？没必要，再找一个人，让他把自己归入这4类，一定就可以吗？他人可以臆断你属于哪一类，但是自己的认可与他人的承认是不同的。

在这样一个奇怪的位置，疲劳，惶恐。前途未卜。学太久了，没有任何动力；玩太久了，深深懊悔。
所以

Belong to NONE.

逃避现实，也奔向事实。

THE END
