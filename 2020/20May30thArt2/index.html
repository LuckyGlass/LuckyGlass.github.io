еӯҰд№ з¬”и®° - еӣһж–ҮиҮӘеҠЁжңәPAM

еҸ‘иЎЁдәҺ 2020-05-30 жӣҙж–°дәҺ 2020-06-01 еҲҶзұ»дәҺ еӯҰд№ з¬”и®° зғӯеәҰпјҡ Valineпјҡ

дёҖзұ»иғҪеӨҹжҺҘеҸ—дёҖдёӘдё»дёІеҶ…зҡ„жүҖжңүеӣһж–ҮеӯҗдёІзҡ„иҮӘеҠЁжңә~

# жҰӮеҝө

еӣһж–ҮиҮӘеҠЁжңәPAMйқһеёёзұ»дјјдәҺеҗҺзјҖиҮӘеҠЁжңәSAMпјҢжҲ–и®ёжҳҜеҖҹйүҙдәҶSAMзҡ„зҒөж„ҹгҖӮ

е…·дҪ“ең°иҜҙпјҢPAMзҡ„жҜҸдёӘиҠӮзӮ№д»ЈиЎЁдёҖдёӘеӣһж–ҮдёІпјҢе№¶дё”иҠӮзӮ№пјҲйҷӨдәҶиө·е§ӢиҠӮзӮ№пјүдёҺдё»дёІдёӯзҡ„жүҖжңүеӣһж–ҮеӯҗдёІдёҖдёҖеҜ№еә”пјҢд№ҹе…·жңүдёӨз§Қиҫ№пјҡ

  1. иҪ¬з§»иҫ№ next[c]пјҢжҢҮеҗ‘е°ҶеҪ“еүҚиҠӮзӮ№д»ЈиЎЁзҡ„еӣһж–ҮдёІзҡ„дёӨз«ҜеҠ дёҠеӯ—з¬Ұ cеҫ—еҲ°зҡ„еӣһж–ҮдёІзҡ„иҠӮзӮ№гҖӮ
  2. еҗҺзјҖиҫ№ linkпјҢжҢҮеҗ‘еҪ“еүҚиҠӮзӮ№д»ЈиЎЁзҡ„еӣһж–ҮдёІзҡ„жңҖй•ҝзҡ„еӣһж–ҮзңҹеҗҺзјҖзҡ„иҠӮзӮ№гҖӮ

жүҖжңүиҪ¬з§»иҫ№жһ„жҲҗдёҖдёӘ DAGпјҢжүҖжңүеҗҺзјҖиҫ№жһ„жҲҗдёҖжЈөж ‘гҖӮ

дҪҶжҳҜдёҺSAMдёҚеҗҢпјҢPAMжңүдёӨдёӘиө·е§ӢиҠӮзӮ№ 0 е’Ң -1гҖӮеӣ дёәеӣһж–ҮдёІж №жҚ®й•ҝеәҰзҡ„еҘҮеҒ¶еҸҜд»ҘеҲҶдёәдёӨзұ»пјҢйӮЈд№Ҳе°ұзӣёеә”ең°вҖ”вҖ”еңЁ 0 зҡ„дёӨз«ҜеҠ еӯ—з¬ҰзӣёеҪ“дәҺжҳҜдёҖдёӘеҒ¶еӣһж–ҮдёІпјҢиҖҢеңЁ -1 дёӨз«ҜеҠ еӯ—з¬ҰзӣёеҪ“дәҺеҫ—еҲ°дёҖдёӘеҘҮеӣһж–ҮдёІгҖӮ

йҷӨжӯӨд№ӢеӨ–пјҢжҜҸдёӘиҠӮзӮ№иҝҳз»ҙжҠӨдәҶдҝЎжҒҜ $len$пјҢеҚіеҜ№еә”зҡ„еӣһж–ҮдёІзҡ„й•ҝеәҰгҖӮ

# жһ„йҖ 

з»ҷе®ҡдё»дёІ $S$пјҢиҰҒжұӮжһ„йҖ  $S$ зҡ„PAMгҖӮ

иҝҷйҮҢжҜ”SAMиҰҒз®ҖеҚ•еҘҪеӨҡ

еҲқе§ӢеҢ–пјҢдёҚйҡҫзҗҶи§Јпјҡ0 зҡ„ $len$ дёә $0$пјҢ-1 зҡ„ $len$ дёә $-1$гҖӮз„¶еҗҺ 0 зҡ„ link жҢҮеҗ‘ -1пјҢеҸҜд»Ҙиҝҷж ·зҗҶи§Јпјҡ0 д»ЈиЎЁвҖңеҒ¶з©әдёІвҖқпјҢз”ұдёӨдёӘз©әеӯ—з¬Ұз»„жҲҗпјӣ-1 д»ЈиЎЁвҖңеҘҮз©әдёІвҖқпјҢз”ұдёҖдёӘз©әеӯ—з¬Ұз»„жҲҗгҖӮ

д»Қз„¶дҪҝз”ЁеўһйҮҸжі•пјҢд»ҺдёҠдёҖдёӘдҪҚзҪ®з»“е°ҫзҡ„жңҖй•ҝзҡ„еӣһж–ҮдёІејҖе§ӢпјҢеҗ‘дёӨиҫ№жү©еұ•гҖӮи®ҫеҪ“еүҚиҰҒжҸ’е…ҘдҪҚзҪ®дёә $i$пјҢеҪ“еүҚдҪҚзҪ®зҡ„еӯ—з¬Ұдёә $c$пјҢдёҠдёҖдёӘдҪҚзҪ® $i-1$ зҡ„еҜ№еә”иҠӮзӮ№дёә $las$пјҢеҲқе§Ӣжғ…еҶө $las=-1$гҖӮ

  1. еҰӮжһң $S_{i}=S_{i-len[las]-1}$пјҢд№ҹе°ұжҳҜ $i-len[las]-1$ еҲ° $i$ жҳҜдёҖдёӘеӣһж–ҮдёІпјҢеҲҷж— йңҖж“ҚдҪңпјӣ

  2. еҰӮжһң $S_i\not= S_{i-len[las]-1}$пјҢеҲҷе°Ҷ $las$ з§»еҠЁеҲ° $link[las]$ еҗҺеҶҚжЈҖйӘҢжҳҜеҗҰеҪўжҲҗеӣһж–ҮпјҢеҚіз§»еҠЁеҲ°дёҖдёӘжӣҙе°Ҹзҡ„еӣһж–ҮдёІеҶҚеҲӨж–ӯпјӣжңҖеҗҺдёҖе®ҡдјҡжүҫеҲ°ж»Ўи¶іжқЎд»¶зҡ„ $las$пјҢеӣ дёәдёҖдёӘеӯ—з¬Ұжң¬иә«е°ұжҳҜдёҖдёӘеӣһж–ҮдёІпјҢжӯӨж—¶ $las$ дјҡи·іеӣһ $-1$гҖӮ

  3. еҰӮжһң $next[las][c]$ дёҚдёәз©әпјҢиЎЁзӨәиҝҷдёӘеӣһж–ҮдёІе·Із»ҸеҮәзҺ°иҝҮдәҶпјҢеҲҷ $las$ з§»еҠЁеҲ° $next[las][c]$пјӣ

  4. еҰӮжһң $next[las][c]$ жҳҜз©әпјҢиЎЁзӨәиҝҷдёӘеӣһж–ҮдёІжҳҜз¬¬дёҖж¬ЎеҮәзҺ°пјҢеҲҷж–°е»әиҠӮзӮ№ $cur$пјӣ

    $len[cur]=len[las]+2$пјҢеҚіеңЁеҺҹеӣһж–ҮдёІзҡ„дёӨз«ҜеҠ дёҠеӯ—з¬Ұ $c$пјӣ

    з„¶еҗҺжұӮ $link[cur]$пјҢд»Һ $link[las]$ ејҖе§ӢжІҝзқҖ $link$ иҫ№еҗ‘дёҠи·іпјҢзӣҙеҲ°жүҫеҲ°еӣһж–ҮеҗҺзјҖдёәжӯўпјӣеҰӮжһңжүҫеҲ°дәҶеӣһж–ҮеҗҺзјҖпјҢеҲҷ $link[cur]$ жҢҮеҗ‘иҜҘдҪҚзҪ®пјӣеҰӮжһңжІЎжңүпјҲеҸҜиғҪ $cur$ е°ұжҳҜдёҖдёӘеӯ—з¬Ұжһ„жҲҗзҡ„еӣһж–ҮпјүпјҢеҲҷ $link[cur]$ жҢҮеҗ‘ $0$пјӣ

# ж—¶з©әеӨҚжқӮеәҰ

PAM зҡ„дјҳи¶Ҡзҡ„еӨҚжқӮеәҰеҹәдәҺвҖңдёҖдёӘй•ҝеәҰдёә $n$ зҡ„еӯ—з¬ҰдёІжңҖеӨҡжңү $n$ дёӘжң¬иҙЁдёҚеҗҢзҡ„еӣһж–ҮеӯҗдёІвҖқиҝҷдёӘдәӢе®һгҖӮ

иҜҒжҳҺжҜ”иҫғз®ҖеҚ•пјҢеҸӘйңҖиҰҒиҜҒжҳҺеҗ‘еҺҹеӯ—з¬ҰдёІзҡ„жң«е°ҫж·»еҠ дёҖдёӘеӯ—з¬ҰпјҢжңҖеӨҡеҸӘдјҡеўһеҠ дёҖдёӘжң¬иҙЁдёҚеҗҢзҡ„еӣһж–ҮеӯҗдёІгҖӮ

еңЁеҺҹеӯ—з¬ҰдёІжң«е°ҫеҠ дёҠдҪҚзҪ® $i$пјҢеҲҷйҷӨдәҶд»Ҙ $i$ з»“е°ҫзҡ„жңҖй•ҝзҡ„еӣһж–ҮеӯҗдёІпјҲеӣҫдёӯй»„иүІзүҮж®өпјүеҸҜиғҪжҳҜз¬¬дёҖж¬ЎеҮәзҺ°пјҢе…¶д»–д»Ҙ $i$ з»“е°ҫзҡ„еӣһж–ҮеӯҗдёІпјҲжҜ”еҰӮз»ҝиүІзүҮж®өпјүдёҖе®ҡеҮәзҺ°иҝҮпјҢеӣ дёәеҸҜд»ҘеҜ№з§°иҝҮеҺ»пјҲд»Ҙй»„иүІзүҮж®өзҡ„дёӯзӮ№еҜ№з§°пјүжүҫеҲ°дёҖдёӘдёҖж ·зҡ„еӯ—з¬ҰдёІгҖӮ

йӮЈд№ҲпјҢPAM зҡ„зӮ№ж•°е°ұеҸӘжңү $n+2$ дёӘпјҲдёӨдёӘиө·е§ӢзӮ№пјүпјҢз©әй—ҙеӨҚжқӮеәҰ $O(n)$пјҢlinkж ‘зҡ„ж·ұеәҰиҫҫеҲ° $O(n)$гҖӮ

еўһйҮҸжі•жһ„йҖ зҡ„ж—¶й—ҙеӨҚжқӮеәҰд№ҹжҳҜ $O(n)$пјҢеҲҶжһҗеўһйҮҸжі•дёӯдёҖзӣҙеңЁз§»еҠЁзҡ„ $las$ жҢҮй’Ҳпјҡ

  1. $las$ еҸӘдјҡжІҝзқҖ $next$ иҫ№иө° $n$ ж¬ЎпјҢжҜҸж¬Ў $len[las]$ еўһеҠ  $1$пјӣ
  2. $las$ жІҝзқҖ $link$ иҫ№иө°пјҢ$len[las]$ дёҘж јеҮҸе°ҸгҖӮ

йӮЈд№Ҳз”ұдәҺ $len[las]\ge-1$пјҢеҲҷжІҝзқҖ $link$ иҫ№иө°зҡ„ж¬Ўж•°дёә $O(n)$пјҢеҲҷж—¶й—ҙеӨҚжқӮеәҰд№ҹдёә $O(n)$гҖӮ

# еұҖйҷҗ

иҷҪз„¶ PAM йқһеёёдјҳз§ҖпјҢдҪҶжҳҜе®ғиҝҳжҳҜдёҚиғҪе®Ңе…ЁйЎ¶жӣҝ Manacharпјҡ

  1. PAM зҡ„з©әй—ҙејҖй”Җе·ЁеӨ§пјҢеӣ дёәеӯ—з¬ҰйӣҶеӨ§е°ҸдёҖиҲ¬жҳҜ 26пјҢе…үжҳҜ next ж•°з»„е°ұиҰҒејҖ $26\times n$ иҝҷд№ҲеӨҡпјӣйҷӨйқһж—¶й—ҙжҚўз©әй—ҙпјҢз”Ё hash еӮЁеӯҳеӣҫпјҲеӣ дёә next иҫ№зҡ„ж•°йҮҸеҸӘжңү $O(n)$пјү
  2. PAM жұӮзҡ„жҳҜд»ҘдёҖдёӘдҪҚзҪ®з»“е°ҫзҡ„еӣһж–ҮдёІпјҢиҖҢ Manachar жұӮзҡ„жҳҜд»ҘдёҖдёӘдҪҚзҪ®дёәдёӯзӮ№зҡ„еӣһж–ҮдёІпјҢжңүдәӣйўҳз”Ё PAM е°ұжҜ”иҫғйә»зғҰ
